# leak-detector package

Leak Detection.

[![Gyazo](http://i.gyazo.com/9365eebd68cbfe51b3ba4be4a3992de5.png)](http://gyazo.com/9365eebd68cbfe51b3ba4be4a3992de5)

## Settings

* `maxDisposables` (default:200)
* `maxMarkers` (default:200)

## Commands

* `leak-detector:start`
* `leak-detector:stop`

## Keymap

No keymap by default.
